# G - GCD Permutation

## Source
- URL: https://atcoder.jp/contests/abc230/tasks/abc230_g

## Summary
- 길이 N의 순열 P가 주어질 때, GCD(i,j) ≠ 1 이고 GCD(P_i, P_j) ≠ 1 을 동시에 만족하는 쌍 (i,j) (1≤i≤j≤N)의 개수를 구하라.

## Constraints
- 1 ≤ N ≤ 2×10⁵
- P는 1~N의 순열
- Time limit: 5초

## Input / Output Notes
- 입력: 정수 N, 이후 N개의 정수로 이루어진 순열 P
- 출력: 조건을 만족하는 쌍의 수 (단일 정수)
